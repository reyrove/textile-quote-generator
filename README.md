# Textile Quote Generator

A simple, elegant web application that generates random inspirational quotes about textiles, fashion, and design. Perfect for designers, artists, and anyone seeking creative inspiration.

## Features

- Random quote generation with each click
- Beautiful textile-inspired color palette
- Responsive design for all devices
- Curated collection of fashion and textile quotes
- Smooth hover effects and transitions

## Quote Collection

The app includes quotes from:

- Coco Chanel
- Zandra Rhodes
- Rachel Zoe
- Ralph Lauren
- Giorgio Armani
- And more textile wisdom

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/reyrove/textile-quote-generator.git
   ```

2. Navigate to the project folder
   ```bash
   cd textile-quote-generator
   ```

3. Open `index.html` in your browser

## Usage

1. Open the application in any modern web browser
2. Click the "New Quote" button
3. A random textile-inspired quote will appear
4. Repeat to discover more wisdom

## Browser Support

- Google Chrome
- Mozilla Firefox
- Safari
- Microsoft Edge

## Customization

You can easily add more quotes by editing the `quotes` array in `script.js`:

```javascript
const quotes = [
  {
    text: "Your quote here",
    author: "Author name"
  }
];
```
